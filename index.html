Knowhow Cloud Computing

Cloud Computing sind service, welche Online angeboten werden. Statt dass man seinen ganzen Speicher Physisch besitzt kann man den Speicher Online Kaufen und dann ist dieser Online verwendbar.

Cloud Computing verwendet Remoteressourcen. Dadurch sparen Unternehmen Kosten für Server und weitere Geräte.

Cloud Computing Beispiele sind folgende:
  • Microsoft
  • Icloud
  • Virtuell
  • Ressourcen
  • Server
  • Storage
  • Datenbanken
  • Networking
  • Software

Vorteile

  • Anbieter kümmern sich um die Daten
  • Datensicherung ausserhalb von zu Hause
  • Zusammenarbeit mit anderen möglich
  • Theoretisch unbegrenzt Speicherplatz
  • Keine Hardware Kosten

Nachteile

  • Nur möglich mit Internetverbindung
  • Zum teil Geschwindigkeitsprobleme
  • Abhängigkeit
  • Schutz der gespeicherten Daten
  • Mangel an Anpassungsmöglichkeiten
